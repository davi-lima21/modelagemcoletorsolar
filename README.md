# Simulador de Coletor Solar

Este projeto é uma interface gráfica e simulador desenvolvido em Java para modelar o funcionamento de coletores solares. Ele utiliza cálculos baseados em artigos científicos e conceitos de controle de processos para analisar a dinâmica térmica do sistema, permitindo o estudo da eficiência de coletores solares em diferentes condições ambientais.

## Funcionalidades
- Simulação do funcionamento de coletores solares baseada em modelos de controle de processos.
- Interface gráfica interativa para visualização de dados.
- Gráficos que exibem a variação de temperatura ao longo do tempo.
- Configuração de parâmetros como temperatura ambiente e fluxo do fluido.

## Objetivo
Fornecer uma ferramenta educacional e de pesquisa para analisar o desempenho de sistemas solares térmicos, com base em métodos de modelagem computacional e controle de processos.

## Tecnologias Utilizadas
- **Linguagem:** Java
- **Interface Gráfica:** Swing
- **Cálculos Baseados em:** Controle de processos e artigos científicos
- **Build Tool:** Maven
- **Controle de Versão:** Git

## Aplicações
- Ensino e pesquisa sobre sistemas solares térmicos.
- Análise de eficiência energética de coletores solares.
- Simulações computacionais de sistemas térmicos.
